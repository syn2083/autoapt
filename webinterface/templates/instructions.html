Instructions

General Information:


Within the installation directory for this application is a folder called demo_setup_files. This folder contains the following sub folders:

  • APTImports: This folder contains the user/device files to use for server import and setup.
  • APTScripts: This folder contains the Python scripts APT Server requires to pass job details to this application. (Auto-Installed)
  • CleanDB: This folder contains a MSSQL 2014 clean Database, pre-configured with all users/devices/alerts/icd setups in use for this demo.

Configuration files for this demo are located in the config_files directory of the installation directory. You will need to close the application and re-open to reload configurations. All local paths for APT are held here.

The installer for this demo system comes equipped with a Clean DB for your use, APT_AutoDemo-MSSQL14. The server will also require you to import the users and devices export files also included in the demo_setup_files folder of the installation directory if not using the starter DB.
